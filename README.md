# 📊 Loan Risk Analysis Dashboard

An interactive Loan Risk Analysis Dashboard built using Microsoft Power BI to analyze credit risk, loan approvals, and customer risk patterns.

---

## 📸 Dashboard Preview

![Loan Risk Analysis Dashboard](Loan_Risk_Dashboard.png)

---

## 📌 Project Overview

This project analyzes loan applications and customer credit risk using Microsoft Power BI. The dataset was cleaned in Microsoft Excel, analyzed using SQL queries in SQL Server Management Studio (SSMS), and visualized through an interactive Power BI dashboard to identify risk patterns based on employment status, loan purpose, credit utilization, bankruptcy history, and previous loan defaults.

---

## 🛠️ Tools Used

---

## 📂 Dataset

- Loan Risk Dataset (Cleaned CSV)
- The dataset was cleaned in Microsoft Excel before importing into SQL Server Management Studio (SSMS) and Power BI for analysis.

---

## 📈 Dashboard Features

- Total Applications KPI
- Approved Rate
- Average Risk Score
- High Risk Customers
- Average Loan Amount
- Loan Risk Category Distribution
- Average Risk Score by Loan Purpose
- Debt-to-Income Ratio Impact on Risk Score
- Risk Score by Credit Utilization Group
- Previous Loan Defaults Impact on Risk Score
- Bankruptcy History Impact on Risk Score
- Average Risk Score by Employment Status
- Interactive Filters (Employment Status, Loan Purpose, Home Ownership Status)

---

## 💡 Key Insights

- Total loan applications analyzed: 20K.
- Overall loan approval rate is 23.9%.
- The average customer risk score is 51.
- High-risk customers account for 160 records.
- Bankruptcy history is associated with higher average risk scores.
- Customers with previous loan defaults have higher risk scores.
- Risk score varies across different loan purposes and employment statuses.
- Interactive filters allow detailed analysis by employment status, loan purpose, and home ownership status.
